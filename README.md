# HarmoniSync

A cross-device audio distribution application enabling synchronized listening experiences.

## Features
- **Host/Listener Architecture**: One device broadcasts, others listen.
- **Low Latency**: Optimized for real-time synchronization.
- **Web Interface**: Accessible via browser on mobile and desktop.
- **Room Security**: Host-controlled access with waiting rooms.

## Tech Stack
- Frontend: HTML/CSS/JS (or Framework if applicable)
- Backend: Node.js / Socket.io (Assumed based on real-time needs)

## Installation
1.  `npm install`

## Usage
1.  `npm start`
